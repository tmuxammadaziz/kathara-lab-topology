# Kathara Network Topology Lab

## Overview
This project is a network topology built using Kathara, a network emulation tool based on Linux containers. The goal of this lab is to practice and understand basic networking concepts such as routing, IP configuration, and connectivity between multiple nodes.

## Features
- Custom network topology design
- Configuration of routers and hosts
- IP addressing and routing setup
- Connectivity testing using ping and traceroute
- Linux-based virtual network environment

## Tools & Technologies
- Kathara
- Linux (Debian-based environments)
- Networking concepts (IP, routing, subnets)
- CLI tools (ping, traceroute, ifconfig/ip)

## Project Structure
- lab.conf — topology configuration file  
- startup scripts — node configuration scripts  
- network nodes — routers and hosts setup  

## Purpose
This lab was created as part of my learning process in computer networks and DevOps practices. It helps to understand how real network infrastructures work in a controlled virtual environment.
